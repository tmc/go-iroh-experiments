# go-iroh-experiments

This repository holds experimental modules built on
github.com/tmc/go-iroh. Each experiment is its own Go module so dependencies
stay isolated.

Modules:

- `pkarrnaming`: IPNS-style content naming over pkarr relay transport.
- `contentdiscovery`: planned.
- `dagsync`: planned.
- `s3baostore`: planned.
- `h3iroh`: planned.
